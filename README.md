# Weather_Forecast
Demo: <a href="https://youtu.be/4SDlc_k94dU"> Demo Video</a>
* This is a native app based on Kotlin and use the OpenWeatherMap API to pick up data from their server.
Including:
- Kotlin
- Coroutine
- Retrofit + Gson
- MVVM Structure
- LiveData
